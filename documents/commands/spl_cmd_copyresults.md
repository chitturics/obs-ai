 ## Syntax
copyresults <copyresults-dest-option> <copyresults-sid-option> dest=<string> sid=<string>

## Description
Copies the results of a search to a specified location within the config directory structure. This command is primarily used to populate lookup tables. The destination file where to copy the results to. The string is interpreted as path relative  The search id of the job whose results are to be copied. Note, the user who is running this
