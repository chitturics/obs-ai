 ## Syntax
oldreturn (<int:count>)? (<field:alias>=<field>)* (<field>)* ($<field>)*

## Description
A convenient way to return values from a subsearch. The 'oldreturn' command is a deprecated version of the 'return' command. Use it only if you are having problems with 'return'. Go to 'return' for the full description of this command.
