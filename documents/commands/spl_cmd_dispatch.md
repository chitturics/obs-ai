 ## Syntax
dispatch (ttl=<num>)? (maxresults=<num>)? (maxtime=<num>)? (id=<string>)? (spawn_process=<bool>)? (label=<string>)? (start_time=<num>)? (end_time=<num>)? <server-list> [<search-pipeline>] (<string> )*

## Description
Encapsulates long running, streaming reports. \i A list of possibly wildcarded servers changes in the context of the differences. Try it see if it makes sense.
